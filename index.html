About
PhillyBusFinder is a tool that helps Philadelphia residents track SEPTA bus locations in realtime. This is accomplished by fetching open bus and schedule data to create an improved API and user experience. This project was created at Philly Codefest 2014. This project is no longer live, but you can view our code at GitHub.
